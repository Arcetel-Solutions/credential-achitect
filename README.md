# Credential Architect

**Credential Architect** is a free, lightweight desktop application for **bulk generation of vouchers, credentials, or tokens** from CSV or spreadsheet data.

It is designed to be **simple, fast, offline**, and **highly customizable**, with no installation, no login, and no internet connection required.

Developed and maintained by **Arcetel Solutions Private Limited (Zimbabwe)**.
