# Credential Architect

**Credential Architect** is a free, lightweight, cross-platform credential and voucher generation system developed by **Arcetel Solutions Pvt Ltd (Zimbabwe)**.

It enables fast, professional generation of **print-ready PDF vouchers** such as:
- Wi-Fi access vouchers
- Login credentials
- Event access tokens
- Serial numbers and ID cards
- Institutional or internal credentials

The application is designed to be **simple, portable, and installation-free**.
